#  Project Management App

A full-stack Project Management Application designed to help teams efficiently manage projects, tasks, notes, and collaboration workflows.

---

##  Project Status

This project is currently under development.
Core features are functional, and advanced capabilities like **AI integration** and a **Voice Assistant** are planned for upcoming versions.

---

##  Features

* 🔐 User Authentication (Login / Signup)
* 📁 Project Management
* ✅ Task & Subtask Handling
* 📝 Notes System
* 👥 Team Collaboration
* ☁️ File Upload (S3 Integration)

---

##  Planned Features (Upcoming)

The following features are **not implemented yet**, but are part of the future roadmap:

* 🤖 AI-powered Assistant
* 🎤 Voice Command Agent
* 🧠 Task and Note Summarization using AI
* 📊 Smart Project Insights

### 💡 Example (Planned Functionality)

> "Create a task for UI design by tomorrow"

The system will interpret the command and automatically create a task.

